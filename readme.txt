Nom du projet : CY-MEMORY-RPG


Projet fin d'annee CY-TECH, preing-1
Participants: Damien MOUCHARD, Evan SZPOPER, Gabriel LEGROS
DATE DE RENDU : 27/05/2023

Pour lancer le projet :
- Ecrire make dans la console
- Ecrire ./exec dans la console


Quand vous lance le programme, vous arrivez sur le menu :
- '1' pour lancer une partie
- '2' pour regarder les regles
- '3' pour regarder les statistiques des anciennes parties
- '4' pour quitter le jeu

Si vous entrer dans les regles du jeu, vous pouvez appuyer sur :
- 'm' pour revenir au menu principale
- 'q' pour quitter la partie

- '2' pour revenir au menu principal


REGLES :
Le but du jeu est de recuperer une arme speciale et un coffre legendaire.
L'arme speciale est donne en fonction du role des joueurs.

L'eppe de feu est pour le guerrier
Le baton de controle des familiers est pour le ranger
Le grimoire interdit pour le mage
La dague de sommeil pour le voleur

Pour choisir une arme, il fait entrer :
- '1' pour choisir la torche
- '2' pour choisir le miroir
- '3' pour choisir la hache
- '4' pour choisir la l'arc

La torche permet de tuer le zombie
Le miroir permet de tuer le basilique
La hache permet de tuer le troll
L'arc permet de tuer la harpy

Les deplacements du joueur :
- 'z' pour aller vers le haut
- 'q' pour aller vers la gauche
- 's' pour aller vers le bas
- 'd' pour aller vers la droite
